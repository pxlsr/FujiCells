The chip being reverse engineered is the Roland MB87419 DPCM DSP.

CMOS layout mapping and ASIC reverse engineering project.\
Please see the SVG for the data, and the PDF for the cell descriptions.\
The PDF will be updated in due time to reflect their CMOS logic type.
## Current Status
I am currently in the process of rigorous verification, along with the renaming of vector objects to their respective cell name.\
It is being done layer by layer (SRAM1 and SRAM2, SidewaysArea, and Adder are all verified, and I am currently doing the standard cells.)\
It has taken me over 800 hours by this point to get the file to where it is.\
